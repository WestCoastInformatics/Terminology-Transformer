Introduction

The NDC-RXNORM transformer application provides an API and basic user interface for interacting with RXNORM concepts, NDC (National Drug Codes), and SPL (Standard Product Label) Set Ids. The tool is loaded with all versions of RXNORM dating back to June 2007 (the first appearance of NDC code data).

REST API

The REST API comprises six functions, four of which represent the core services and two of which are used to support the user interface. The full documentation for the API can be found by clicking the API link in the header . Following is a description of the core services.

  • Get NDC Info
    • Takes an NDC as a parameter and returns a document containing the current RXNORM concept, its active status, the current SPL Set Id, and the RXNORM concept history.
  • Get RXCUI Info
    • Takes an RXCUI (an RXNORM concept id) as a parameter and returns a document containing the RXNORM concept info, its active status, any associated SPL Set Ids, and a complete history of NDCs associated with it.
  • Get NDC Properties
    • Takes an NDC as a parameter and returns a document a containing details of the NDC, including various properties associated with it in RXNORM.
  • Get NDC Properties for SPL Set Id
    • Takes an SPL Set Id as a parameter NDC as a parameter and returns a document containing details of all of the NDCs associated with it.

Sample Invocations

Following are examples of calls that demonstrate the different ways of performing searches in this application.

  • Searching by name of a medication, e.g. "Aspirin"
  • Searching by RXNORM concept, e.g. "351772"
  • Searching by NDC, e.g. "00143314501"
  • Searching by SPL Set Id, e.g. "8d24bacb-feff-4c6a-b8df-625e1435387a"
